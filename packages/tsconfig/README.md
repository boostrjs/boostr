# @boostr/tsconfig

Default TypeScript configuration used by Boostr.

## Installation

```bash
npm install @boostr/tsconfig --save-dev
```

## Usage

At the root of your project, create a `tsconfig.json` file with the following content:

```json
{
  "extends": "@boostr/tsconfig",
  "include": ["src/**/*"]
}
```
